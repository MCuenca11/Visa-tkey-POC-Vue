# vue-app

## Create a local build of tKey
```
yarn
yarn bootstrap
yarn pack:lerna
```

## Project setup

Update `tkey/*` dependencies' version.

```
npm install
npm run install:local
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
